→ Provides link + QR code to restaurant owner (e.g., via /admin)

→ Opens browser-based page at /order/<id>
→ Can activate web push notifications (1 click)
→ No login, no name – identification is only via the link

Restaurateur controls completion
→ Can either:
a) Trigger manually (immediate notification)
b) Enter exact completion date (ISO format such as 2025-12-02T14:30:00)
→ System sends a web push notification at the specified time or when triggered manually

System supports multiple orders simultaneously
→ Each order has its own status (ID, time, push subscription, status)

app.py → Flask backend with the following routes:
POST /create-order → creates new order + QR link
